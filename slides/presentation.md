# CougPulse

Campus awareness and safety intelligence for WSU

---

## What It Does

- Maps real-time room noise for students
- Gives security a device and alert operations console
- Supports face enrollment and incident review

---

## Core Experiences

### Student View

- Heatmap of quieter classrooms
- Fast room-by-room campus visibility

### Security View

- Device management
- Floor planning
- Alert review

---

## Stack

- Next.js
- Prisma + PostgreSQL
- Tailwind CSS
- face-api.js
- Marp for presentation authoring
